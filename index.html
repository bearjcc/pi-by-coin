Estimating π with a Coin

Flip until heads lead, record the fraction. The average tends to π/4.

Did you know that you can estimate the value of pi by flipping coins? We flip until heads first lead, then record the fraction of heads at that moment. If you do many runs and average those fractions, that average gets closer and closer to π/4, so 4×(average) estimates π. Try it with the simulator below; learn why it works here.

Why does this work?

The game

We use a fair coin: each flip is heads with probability 1/2 and tails with probability 1/2, and flips are independent (no flip influences another). We stop the first time the total number of heads is greater than the total number of tails, and we record the fraction of flips that were heads at that moment (e.g. 3 heads out of 5 flips gives 3/5). That fraction is always greater than 1/2.

Why a fixed number?

When you do many runs and average those fractions, the average stabilizes: it tends to one fixed number. So we get a reliable estimate by repeating the process.

A useful picture: the random walk

Think of each flip as a step: heads = +1, tails = −1. We start at 0 and add these steps. So we are watching a “random walk” that goes up or down by 1 each time. We stop the first time the total is positive (heads have just taken the lead). That moment is the end of the run.

When we stop

Because we go up or down by 1 each step, we can only go from 0 to positive in one step: we must have one more head than tail. So we can never stop after an even number of flips (you can check: 0, 2, 4, … flips always leave heads and tails tied or tails ahead). We stop only after 1, 3, 5, 7, … flips. The number of flips when we stop is called the stopping time (often written τ).

Who counts the paths? Catalan numbers

The number of ways to stop at 3 flips, 5 flips, 7 flips, … is given by the Catalan numbers. Each run that stops at flips corresponds to a path that stays at or below zero for the first steps and ends at zero (tails never lead), then one head. The Catalan number counts exactly those paths.
